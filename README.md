# Automated Mapping of Dry Stone Walls in Terraced Landscapes Using Airborne Laser Scanning Data

This repository accompanies our paper on automated dry stone wall mapping in terraced landscapes from ALS-derived terrain data. It provides the code for reference and feature generation, preprocessing, evaluation, and area-wide inference and mapping. The deep learning model architecture and training code are available in the separate repository: https://github.com/smfloery/landformAI
